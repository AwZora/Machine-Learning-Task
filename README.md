# Machine Learning Task

A collection of Machine Learning assignments covering fundamental concepts, practical implementations, linear algebra, and statistics using Python and Jupyter Notebook.

## Contents

| Assignment | Topic |
|------------|--------|
| Tugas 1 | Introduction to Machine Learning with Python |
| Tugas 2 | Scikit-Learn Cookbook |
| Tugas 3 | Practical Linear Algebra |
| Tugas 4 | Practical Statistics for Data Scientists |

## Structure

Each project contains:
- Theory and explanations
- Code implementation
- Analysis and discussion
- Conclusions

## Tools & Libraries

- Python
- Jupyter Notebook
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-Learn

## Learning Objectives

- Understand Machine Learning fundamentals
- Apply data preprocessing techniques
- Implement Machine Learning algorithms
- Explore linear algebra concepts for ML
- Apply statistical methods in data analysis
